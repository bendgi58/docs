# Добро пожаловать в my10XsalesAI

Платформа для интеллектуальной автоматизации продаж и интеграции CRM.

## Начало работы

**Редактируйте свою документацию**
Настройте документацию локально для удобной разработки. Синхронизируйтесь с GitHub за один клик.

**Предпросмотр изменений**
Проверьте свои изменения перед публикацией, чтобы убедиться в их безупречности.

---

## Сделайте его своим

Обновите документацию в соответствии с вашим брендом и добавьте ценный контент для оптимальной конверсии пользователей.

**Кастомизируйте стиль**
Настройте документацию под цвета и бренд вашей компании.

**Справочник API**
Автоматически генерируйте документацию эндпоинтов из OpenAPI спецификации.

**Добавляйте компоненты**
Используйте компоненты для создания богатых документов.

**Продвинутые настройки**
Исследуйте расширенные параметры конфигурации.
